# yFullScreen

Unity Editor の Game View を再生中にフルスクリーン表示するユーティリティ。F12 または指定キーでトグル。新旧 Input System どちらにも対応（追加依存なし）。

## インストール

1. Unity で **Window → Package Manager** を開く
2. 左上の **+** をクリック → **Add package from git URL**
3. 以下を入力して **Add** をクリック

```
https://github.com/YProjectGit/yFullScreen.git
```

## 使い方

- **F12** または設定したキーで、再生中の Game View のフルスクリーン表示をオン/オフできます。
- サンプルの Prefab を Import すると、`Assets/Samples/yFullScreen/FullScreen` 以下に配置されます。

## 要件

- Unity 6000.0 以降
