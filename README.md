# yFullScreen

**プロジェクトを再生しているとき、GameViewをフルスクリーン（画面いっぱい）で表示できるようにするパッケージです。**

- プレイ中の画面を大きく見たいときや、実機に近い見え方で確認したいときに便利です。
- **F12 キー**で、フルスクリーン ⇔ 通常表示を切り替えられます。

---

## このパッケージでできること

- **フルスクリーン表示** … GameViewを、モニター全体に広げて表示できます。
- **F12 で切り替え** … 再生中に F12 を押すだけで、フルスクリーンと通常表示を切り替えれます。（キーの割当ては変更できます）

---

## インストール方法（初めての方向け）

Unity に「パッケージ」として追加します。次の手順をそのままやってください。

### ステップ 1：Package Manager を開く

1. Unity エディタの一番上のメニューから **Window**（ウィンドウ）をクリック
2. 一覧の中から **Package Manager**（パッケージ マネージャー）をクリック
3. 新しいウィンドウが開きます。これが「パッケージを追加・管理する画面」です

### ステップ 2：Git URL からパッケージを追加する

1. Package Manager のウィンドウで、左上あたりにある **+**（プラス）ボタンをクリック
2. 出てきたメニューから **Add package from git URL...** をクリック
3. テキストを入力する欄が表示されるので、次の URL を**そのままコピー＆ペースト**します

   ```
   https://github.com/YProjectGit/yFullScreen.git
   ```

4. **Add**（追加）ボタンをクリック
5. 少し待つとインストールが完了します。Package Manager の一覧に **yFullScreen** が出ていれば成功です

> **補足**：「Add package from git URL」は、GitHub などにあるパッケージを URL で直接追加する方法です。Unity 標準のメニューにないパッケージを入れるときによく使います。

## 使い方

1. **サンプルの「yFullScreen」のPrefab をシーンに配置する**  
   サンプルを Import すると `Assets/Samples/yFullScreen` に **yFullScreen** の Prefab が置かれます。その Prefab を、制作中の Unity シーンの **Hierarchy ビュー** にドラッグ＆ドロップします。

2. **いつも通りプロジェクトを再生する**  
   Unity で **▶ 再生ボタン**を押して、Play モードにします。

3. **F12 を押す**  
   再生中の GameViewが、画面いっぱいのフルスクリーンで表示されます。

4. **もう一度 F12 を押す**  
   通常の GameView表示に戻ります。

この「F12 でフルスクリーン ⇔ 通常」の切り替えを、再生中は何度でもできます。

---

## その他の設定

シーンに配置した **yFullScreen** の Prefab を選択すると、Inspector で次の項目を変更できます。

- **FullScreen on Play**  
  再生開始時に GameView を自動でフルスクリーンにするかどうか。オンにすると、再生ボタンを押した時点でフルスクリーンになります。

- **FullScreen Key**  
  フルスクリーン切り替えに使うキー。デフォルトは **F12** です。ボタンを押して別のキーを割り当てられます。制御・修飾キー（Shift や Ctrl など）は指定できず、その場合は F12 が使われます。

---

## 注意事項

このパッケージのフルスクリーン機能は、**エディタの Play モード専用**です。ビルドした実行ファイル（プロジェクター）では動作しません。

---

## トラブルシューティング

**フルスクリーンから戻れない**

1. もう一度 **F12**（または設定したキー）を押して、通常表示に戻れるか試してください。
2. 反応しない場合は、**Unity エディタを強制終了**してください。  
   - Windows: タスクマネージャーで「Unity」を選択し「タスクの終了」  
   - macOS: 「Unity」を選択して Command + Q、または Dock から「終了」を長押し  
   未保存のシーンやスクリプトがある場合は、強制終了で失われることがあります。可能な範囲で保存してから終了してください。

- **キーが反応しない**  
  Hierarchy で **yFullScreen** を選択し、Inspector の **FullScreen Key** が押しているキーと一致しているか確認してください。**None** のときは F12 が使われます。**新 Input System** を使っているプロジェクトでは、**Game View にフォーカスがあるときだけ**キーが検知されます。再生中に Game View をクリックしてからキーを押してください。


- **Unity のバージョンや内部 API の違い**  
  まれに挙動が変わる場合があります。
