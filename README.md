# yFullScreen

**プロジェクトを再生しているとき、Game ビューを画面いっぱい（フルスクリーン）で表示できるようにするパッケージです。**

プレイ中の画面を大きく見たいときや、実機に近い見え方で確認したいときに便利です。**F12 キー**で、フルスクリーン ⇔ 通常表示を切り替えられます。

---

## このパッケージでできること

- **Game ビュー** … Unity で「再生ボタン（▶）」を押したときに表示される、プロジェクトの再生画面です。
- **フルスクリーン表示** … その Game ビューを、モニター全体に広げて表示できます。
- **F12 で切り替え** … 再生中に F12 を押すだけで、フルスクリーンと通常表示をトグルできます。
- **追加設定は不要** … 新しい Input System でも古い Input System でも、そのまま使えます。

---

## インストール方法（初めての方向け）

Unity に「パッケージ」として追加します。次の手順をそのままやってください。

### ステップ 1：Package Manager を開く

1. Unity エディタの一番上のメニューから **Window**（ウィンドウ）をクリック
2. 一覧の中から **Package Manager**（パッケージ マネージャー）をクリック
3. 新しいウィンドウが開きます。これが「パッケージを追加・管理する画面」です

### ステップ 2：Git URL からパッケージを追加する

1. Package Manager のウィンドウで、左上あたりにある **+**（プラス）ボタンをクリック
2. 出てきたメニューから **Add package from git URL...** をクリック
3. テキストを入力する欄が表示されるので、次の URL を**そのままコピー＆ペースト**します

   ```
   https://github.com/YProjectGit/yFullScreen.git
   ```

4. **Add**（追加）ボタンをクリック
5. 少し待つとインストールが完了します。Package Manager の一覧に **yFullScreen** が出ていれば成功です

> **補足**：「Add package from git URL」は、GitHub などにあるパッケージを URL で直接追加する方法です。Unity 標準のメニューにないパッケージを入れるときによく使います。

## 使い方

1. **いつも通りプロジェクトを再生する**  
   Unity で **▶ 再生ボタン**を押して、Play モードにします。

2. **F12 を押す**  
   再生中の Game ビューが、画面いっぱいのフルスクリーンで表示されます。

3. **もう一度 F12 を押す**  
   通常の Game ビュー表示に戻ります。

この「F12 でフルスクリーン ⇔ 通常」の切り替えを、再生中は何度でもできます。

### サンプル（Prefab）を使う場合

- Package Manager で yFullScreen を選んだ状態で、右側の **Samples** にある **FullScreen** の **Import** を押すと、サンプル用の Prefab がプロジェクトにコピーされます。
- コピー先は `Assets/Samples/yFullScreen/FullScreen` です。必要に応じてシーンに配置して使えます。
